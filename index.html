Utilize Tab para primeira oitava
Utilize CapsLock para última oitava
Utilize ShiftLeft para recuar a oitava
Utilize ControlLeft para avançar a oitava

Caso queira gravar uma macro, clique em Enable Macro Record para definir a sua macro-list.


Caso queira reproduzir uma macro-list, clique em Run Macro.


Caso queira reproduzir uma macro-list em modo repetição, clique em Repeat Macro.


Caso queira definir um atraso entre repetições, insira o número na caixa de MS Repeat.


1000 representa 1 segundo de atraso entre cada repetição

100 representa 0.1 segundos de atraso entre cada repetição
